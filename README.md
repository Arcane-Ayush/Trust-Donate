# TrustDonate — NGO Accountability Platform

## Project Identity
**Tagline:** A gasless blockchain transparency platform for NGO accountability  
**Chain:** Base Sepolia  
**Gas abstraction:** UGF (Universal Gas Framework)

## Quick Start
1. `cd frontend`
2. `npm install`
3. `npm run dev`

- Person 1: Documentation & Presentation
- Person 2: Frontend Lead (Donor-facing)
- Person 3: Blockchain Lead
- Person 4: Integration & Dashboards
